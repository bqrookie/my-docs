#### 烤鱼

- 原材料
  - 草鱼【**2斤/2人分量**】
- 配菜
  - 豆泡、豆芽、豆皮、土豆
- 调味料
  - 豆瓣酱【**偏咸口，广东微辣**】、盐、酱油、姜葱蒜、香菜
- 做法
  ```
  1.草鱼开背，洗干净，背部开花刀，用少量盐涂抹鱼身【去腥，也可以加少量料酒】

  2.然后热油下锅，煎至两面金黄

  3.煎鱼捞起，不用换锅，直接下姜葱蒜爆香，下豆瓣酱，继续翻炒片刻

  4.下配菜翻炒片刻【配菜看个人喜欢来搭配，这里暂时只下土豆】，然后下适量开水【看鱼的大小和配菜的量来加水】，煮开即可把鱼下锅

  5.鱼差不多熟透的时候把剩余的配菜下锅，出锅前把香菜叶子与聪段撒到表面即可出锅
  ```

- 备注
  ```
  1.辣度可以根据自己口味添加，料头可以添加【小米辣&二荆条】增加辣度

  2.出锅后可以在表面添加干辣椒，热油泼一下增香
  ```
