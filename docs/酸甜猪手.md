#### 酸甜猪手

- 原材料
  - 猪手【半只】
- 配菜
  - 无
- 调味料
  - 白醋、姜葱、冰糖、黄糖、盐、生抽、老抽、话梅
- 做法
  ```
  1.猪手切块，过一遍热水【热水加白醋、姜葱去腥】，猪手捞起洗干净晾干

  2.热锅下油，下点冰糖炒一下糖色【冰糖分量：东鹏饮料的杯子，一杯】,下猪手炒香

  3.然后加调料味【黄糖、盐、生抽、老抽、话梅】 

  4.然后加热水,水没过猪手就够，烧开之后改中火，根据水量来决定关火时间【我这里刚好煮了35分钟，开盖收汁】

  
  ```

- 备注
  ```
  1.黄糖分量【身份证2/3的面积即可】 话梅分量【我这里只下了10粒，味道稍淡】，可以根据自己口味调整
  ```
